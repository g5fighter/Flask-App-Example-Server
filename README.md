# Flask-App-Example-Server
Flask App example to use on your server with the rutes and config files.

# Features
* Functional music download app
* Config files to work perfectly on your Apache Server with an alias

# Important
* Some files are located in the project with an example domain 'miservidor.com'
* The program has been developed in a virtual enviroment
